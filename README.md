# BQuark Starter

Специализированный стартер для BQuark Инкотекс. Программа формирует `START.dat` для BQuark, чтобы опрашивать концентраторы PLC-1 через TCP/IP.

## Назначение

BQuark Starter не является универсальным редактором конфигурации BQuark и не предназначен для коммерческого учета показаний. Его задача — быстро подготовить конфигурационный файл для BQuark, запустить BQuark и определить количество отвечающих счетчиков в заданном комплекте концентраторов.

Показания для учета собираются другой программой.

## Возможности

- редактирование IP и порта TCP/IP;
- ручное добавление от 1 до 15 концентраторов;
- адрес концентратора: 4 символа, первый всегда `2`, остальные символы — цифры или латинские буквы;
- единый непрерывный диапазон PLC-адресов от `1` до `1024`;
- чтение существующего `START.dat` при запуске;
- создание или замена `START.dat` рядом со стартером;
- сохранение существующих строк счетчиков с данными, если концентратор и PLC-адрес остались в новом диапазоне;
- запуск `BQuark.Rev.3.3.exe`;
- запуск стартера с правами администратора.

## Ограничения

Стартер жестко ограничен маршрутом доступа `GPRS/TCP_MODEM` по TCP/IP, концентраторами `PLC_I_CONCENTRATOR` и счетчиками `PLC_I_METER`.

В интерфейсе BQuark оставлена только колонка суммы:

```text
TYPE=COLUMN; WIDTH=200; TITLE=Сумма. кВт*ч; VALUE=BINDATA; FILTER=SUM/00:00
```

`BQuark.Rev.3.3.exe`, справка BQuark и рабочий `START.dat` в репозиторий не включаются.

## Сборка

Требуется Visual Studio с компонентом C++ Desktop Development.

Рекомендуемая конфигурация:

- `Release`
- `x86`
- статическая линковка runtime: `/MT`

Команда сборки для текущей среды:

```powershell
& 'C:\Program Files\Microsoft Visual Studio\18\Insiders\MSBuild\Current\Bin\MSBuild.exe' .\BQuarkStarter.sln /p:Configuration=Release /p:Platform=x86 /t:Rebuild
```

Если используется другая версия Visual Studio, может потребоваться изменить `PlatformToolset` в `BQuarkStarter.vcxproj` на установленный набор, например `v143`.

## Использование

1. Соберите `BQuarkStarter.exe`.
2. Положите рядом со стартером `BQuark.Rev.3.3.exe` или папку `BQuark.Rev.3.3` с файлом `BQuark.Rev.3.3.exe` внутри.
3. Запустите `BQuarkStarter.exe`.
4. Укажите IP, порт, концентраторы и диапазон PLC.
5. Нажмите `Старт`.

Стартер создаст или заменит `START.dat`, затем запустит BQuark.

## Файлы проекта

- `main.cpp` — интерфейс и логика стартера;
- `BQuarkStarter.rc` — ресурс иконки;
- `resource.h` — идентификаторы ресурсов;
- `start.ico` — иконка приложения;
- `START.example.dat` — пример создаваемого файла;
- `BQuarkStarter.sln`, `BQuarkStarter.vcxproj` — файлы Visual Studio.
